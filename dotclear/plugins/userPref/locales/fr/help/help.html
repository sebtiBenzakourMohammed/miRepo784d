User:preferences

Cette page présente des informations techniques brutes généralement gérées par d'autres pages.
Ne modifiez quelque chose que si vous savez ce que vous faîtes.

Par exemple le paramètre nodragdrop se configure sur la page "Mes préférences" dans l'onglet "Mes options".

La page présente deux onglets "Préférences utilisateur·rice·s" et "Préférences globales" qui permettent de gérer (et présenter) respectivement vos préférences et celles communes à tous les utilisateur·rice·s.

Vous pouvez accéder directement aux différentes sections de la page avec le menu de sélection.
